# COLORSLAB

ColorsLab is a LAMP stack web application which allows users to manage and search through a personal list of colors.

## Tech Stack
LAMP (Linux, Apache, MySQL and PHP)

## Setup

1. Clone the repository
2. Ensure `php` and `make` cli tools are installed
3. Ensure mysqli database is setup
4. Run `make serve` to spin up a the application on `localhost:8000`
